# zabbix_templates

Template for **Fujitsu_PRIMERGY_CX2560 SNMP_v3**

Work with zabbix **v3.4**

Read:
* Read Temperature
* Read System Voltage
* Read Fan speeds
* Read System info

Macros:
* {$FAN_SPEED_LIMIT} - default 5000
* {$SNMP_USER} - **_enter your pass_**
* {$SNMP_PASS} - **_enter your login_**
